The Web

What Is WEB?

The Internet is a massive network of networks. It connects millions of computers together globally, forming a network in which any computer can co mmunicate with any other computer as long as they are both connected to the Internet.

The World Wide Web or simply Web is a way of accessing information over the medium of the internet. It is an information-sharing model that is built on top of the Internet (A bunch of connected computers that have websites).

A website can be written using three(3) languages:

  • Html; for marking up the website content
  • CSS (cascading style sheet); for styling the website
  • Scripting language; for making it interactive

A browser:
It is a program that allows you to interact and view contents of the Web (Application programs dedicated for web browsing e.g. opera, Firefox, chrome, Internet Explorer, etc. are called browsers)
Clients:
They are Devices (e.g. laptop, tablet, mobile phones etc.) from which browsers are used.
Servers:
When a computer is connected to the internet and spinning out a website, we call the computer a server because it is serving the website.

How The Web Works

The internet operates on a client / server architecture
-HTTP is a text-based request-response protocol used for web browsing
